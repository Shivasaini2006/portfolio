# Project Images

Store your project screenshots and images here.

## How to Use:

1. Add your images to this folder
2. Push to GitHub
3. Use the raw GitHub URL in your admin panel:
   ```
   https://raw.githubusercontent.com/Shivasaini2006/portfolio/main/public/project-images/YOUR_IMAGE.jpg
   ```

## Example:
If you upload `my-project.jpg` here, use:
```
https://raw.githubusercontent.com/Shivasaini2006/portfolio/main/public/project-images/my-project.jpg
```

## Tips:
- Use descriptive filenames like `ecommerce-app.jpg`
- Recommended image size: 1200x800px or 800x600px
- Keep images under 500KB for faster loading
- Supported formats: JPG, PNG, WebP
